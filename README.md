AI Tutor

A simple RAG-based AI Tutor built using:
- Python
- ChromaDB
- Sentence Transformers (`all-MiniLM-L6-v2`)
- Ollama (`phi3:mini`)

Features:
- PDF text extraction
- Overlapping text chunking
- Semantic search
- Persistent vector database
- ChromaDB integration
- Local LLM integration
- Offline RAG pipeline

Run Project:
python app.py
